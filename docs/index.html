todo

TODO in your terminal

A portable CLI for managing project TODO.md files that lives in your terminal and opens a web dashboard when you need visuals. Actors, comments, status tracking, and two view modes are built in. Written in Rust — one binary, zero dependencies.

CLI + Dashboard

Manage your TODO.md from the terminal, then open a web dashboard for a visual overview. Two modes in one binary.

# #

Tags & Priority

Organise tasks with tags and priority levels (low, medium, high). Filter and search your task list instantly.

@ @

Actors & Comments

Assign actors to tasks and attach comments. Track who did what and keep a running discussion per task.

~>

Kanban Board

Switch between a list view and a kanban board with drag-and-drop. Visualise your workflow at a glance.

^ ^

Status Tracking

Mark tasks as Todo, In Progress, Done, or Blocked with an optional blocked reason. Keep your project moving.

~~$

Shell Completion

Built-in shell completion for bash, zsh, fish, and PowerShell. Tab-complete your way to productivity.
